# Reserve

**Reserve** is a school supply donation platform that connects donors with nearby families and teachers in need.

---

## Features
```
Request & Donation Forms - Submit what you need or what you can give — books, stationery, art supplies, and more
Matching - Our backend algorithm instantly pairs donors with nearby receivers based on location and supply type
One-on-One Chat - Once a match is verified by both parties, a private chat opens to coordinate the transfer
Donation Centers Map - Browse and add nearby donation centers for drop-off or pickup
Amazon Wishlists - Teachers can link their Amazon wishlists to accept direct donations from supporters
```

---

## How It Works

```
1. Submit   →   Fill out a request or donation form
2. Match    →   Algorithm finds a compatible nearby match
3. Verify   →   Both parties confirm and exchange contact info
4. Transfer →   Chat directly to arrange pickup or drop-off
```

## Getting Started

### Prerequisites

- Node.js v18+
- npm v9+
- Git
- Expo Go App (on your phone)

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/claireschair/ReserveApp.git
   cd ReserveApp
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npx expo start
   ```
   Please ensure that your computer and phone are running on the same WiFi.

### Submitting a Donation

1. Log in and navigate to the **Donate** tab
2. Fill in supply details (type, quantity, condition) and your location
3. Submit — the algorithm immediately searches for a nearby match
4. Once both sides verify, a chat thread opens for transfer coordination

### Submitting a Request

1. Log in and navigate to the **Request** tab
2. Fill in supply details (type, quantity, condition) and your location
3. Submit — the algorithm immediately searches for a nearby match
4. Once both sides verify, a chat thread opens for transfer coordination

### Adding and Viewing Donation Drive

1. Log in and navigate to the **Map** tab
2. Click the add donation center button and add the name of the donation center as well as the dates of the donation drive and items avaliable (optional)
3. View the map and see donation drives currently happening

### Linking an Amazon Wishlist (Teachers)

1. Go to your **Profile → Wishlist Integration**
2. Paste your public Amazon wishlist URL and save
3. Your wishlist appears on your public teacher profile for donors to browse
